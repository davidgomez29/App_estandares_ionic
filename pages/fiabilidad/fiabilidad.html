Preguntas Proyecto de Software

Fiabilidad

Madurez:

¿El sitio se muestra como un alojamiento optimo, seguro y llamativo para el cliente por sus caracteristicas?
¿El software cuenta con una documentacion adecuada para su calidad de uso?
¿Existe un documento previo a la realizacion del software basado en su estructura y pautas iniciales?
¿La madurez del software es acorde con las estretegias planificadas de desarrollo en una previa documentacion?
¿Existe una logica entre cada accion realizada por el software?

Tolerancia a fallas:

¿Cuenta el sitio con estandares externos exentos ante fallas alternas que puedan afectar la permanencia del usuario?
¿El software tiene catacteristicas de ajuste según la calidad de graficos de cada dispositivo en que se corra?
¿El software tiene catacteristicas de ajuste según la calidad de procesador de cada dispositivo en que se corra?
¿Cuenta con fallas de lentitud, desorden de graficos, alteraciones en las acciones o botones inservibles?
¿Las fallas del sofware afectan aplicaciones externas?

Recuperacion:

¿Se recupera rapido ante una eventualidad no esperada?
¿El reinicio de la aplicación es optimo y retorna a las caracteristicas iniciales antes del reinicio?
¿La aplicación hace un backup constante sobre sus diversos datos nativos?
