Funcionalidad

Adecuacion:

¿La adecuacion de las acciones brindadas son suficientes para satisfacer el objetivo del software?
¿Existe idoneidad entre la necesidad del cliente y la funcion que otorga el software?
¿Cumple cada boton con su funcion indicada?
 ¿Cuál es la cantidad de objetivos a los que el cliente puede acceder? {{cantobjetivos}} ¿Cuál es la cantidad de objetivos especificados en el software en beneficio del cliente que se cumplen con eficacia? {{cantobjetivoscumplidos}} ¿Cómo es el tiempo de espera de ejecucion de la aplicación? {{nivelesCalidad[tiempoespera] }}

Precision:

¿Es la accion de los botones acorde con la funcion especificada en el boton previamente antes de presionarlo?
¿Los formularios proporcionan informacion suficiente para guiar al cliente en la presicion a la hora de contestar las preguntas?
¿Los datos que solicita el formulario al cliente son precisos con la necesidad del cliente?
¿Los datos de contacto del software cuentan con presicion clara para la visibilidad del cliente?
¿Los botones de acciones se encuentran inmersos con exactitud en las pestañas//paginas correspondientes?
¿Es acorde el software con lo que plantea en su mision a cumplir?
¿Cuál es la calificacion otorgada a la facilidad con la que el cliente puede encontrar con presicion las opciones o peticiones necesitadas? {{cantcali}}

Interoperabilidad:

¿Existe la posibilidad de realizar varias operaciones a traves del proyecto?
¿Es posible conectar hardware externo para aumentar las funciones del software?
¿Es necesaria el uso de aplicaciones externas para el funcionamiento del proyecto?
¿La aplicación permite las conexiones de otras funcionalidades externas pidiendo un previo permiso?
¿Las diversas operaciones del software son suficientes para las acciones que requiere ejecutar el usuario?
¿La aplicación consume información de otro sistema?
¿La aplicación es consumida por otro sistema?

Seguridad:

¿Se ven claras las funciones de seguridad para que el sitio inspire confianza?
¿El uso de la aplicación esta sesgada para algun tipo de usuario especifico o cualquier usuario puede acceder a ella?
¿Hay informacion clara sobre el manejo que van a tener los datos suministrados en la aplicación por el usuario?
¿Los formularios cuentan con aceptacion de permisos de usuario sobre terminos y condiciones?
¿Se requiere de diferentes niveles de autenticación?
¿La capa de información es administrada por un servidor dedicado a parte de la aplicación?
¿La aplicación puede revocar permisos de forma dinámica?
