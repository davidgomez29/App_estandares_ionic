Usabilidad

Facilidad de comprension:

¿Es claro para el cliente el entendimiento de la pagina web para no recurrir a botones alternos buscando lo que requiere?
¿El cliente logra percibir con facilidad cada una de las opciones?
¿La ubicación de los botones es optima para una facilidad de comprension por parte de el cliente?
¿Cada una de las caracteristicas del software estan alineadas para que el usuario final pueda acceder facilmente a sus requerimientos?
¿La aplicación cuenta con ayudas para que el usuario pueda entender lo que no entiende?

Capacidad de aprendizaje:

¿Brinda el proyecto alternativas para mejorar la capacidad de aprendizaje de quien hace uso del mismo?
¿El usuario aprende la aplicación gracias a sus requerimientos?
¿La aplicación puede aprender de los requerimientos del usuario?
¿Las necesidades del usuario son optimas para que la aplicación aprenda de ellas?
¿El usuario puede aprender los procesos ejecutados por la aplicación o son implicitos?

Interfaz Grafica:

¿Existen contrastes o problemas de graficos que distraigan la atencion del cliente?

¿Se puede configurar la calidad de los graficos?

¿La rapidez de la aplicación es optima?
¿Las necesidades del usuario son optimas respecto a lo que realiza la aplicación en funcion de sus requerimientos?
